# pythonJobOffersAgregator
# Opis projektu oraz założenia:
# Projekt mający na celu stworzenie agregatora ofert pracy w pythonie, głównie w celach edukacyjnych, który będzie 
# uruchamiał się cyklicznie, to znaczy codziennie. Przy każdorazowym uruchomieniu komputera, wchodził na konkretne
# strony, i sprawdzał pod kontem nowych ofert pracy, a następnie przechowywał oferty pracy, w celach edukacyjnych,
# w róznych typach baz danych, typu sqlite albo mysql jak równiez w formie tekstowej, binarnej, xml oraz json. 
# Dodatkowo ma być dostępna możliwość przeglądania tych ofert w uproszczonej formie, oraz ewentualny sprawdzian 
# zgodności pomiędzy poszczególnymi
